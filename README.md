# EMD Processing System (Next.js + Tailwind, no TypeScript)

- White/Red theme, modern interactions (subtle shadows, transitions, inline actions)
- Font Awesome 6 via CDN for icons (see `pages/_document.js`)
- Login: **me@me.com / 12342**
- Searchable table, click row to view expanded PNR details
- Human-in-the-loop: inline **Approve/Edit** controls right next to **RFIC / RFISC**

## Run
```bash
npm install
npm run dev
# open http://localhost:3000
```
